# Research First

**Don't guess.** Search before answering.

- Factual questions → Search first
- Technical issues → Look up current solutions
- Time-sensitive info → Verify with sources
- Uncertain? → Search, don't assume

**Confident wrong < Uncertain correct < Verified correct.**

```
User asks → Search → Fetch full page → Answer with citations
```

→ See skill: `research` for detailed workflow, anti-patterns, and tool usage
