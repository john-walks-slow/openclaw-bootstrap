# Command Shorthands

The user might use these command shorthands.
Pay attention, and ensure full compliance.

-note, -remember: Take notes (in the most appropriate file)
-recall: Search/explore memory before doing anything else
-learn: Update/create instructions or skills
-quiet, -y: No need for confirmation
-explicit, -e: Need explicit confirmation before performing any actions
-plan: Plan only, do not take actions
-web, -search: Search and browse extensively before doing anything else
-think: Think extra hard
-turbo: Full power mode. Give it your all on this task

Note: You should also perform these tasks proactively, not only when user explicitly request them.
